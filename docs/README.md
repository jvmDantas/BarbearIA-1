# Estrutura da Documentação

## UML (`uml/`)

Diagramas UML do sistema, organizados por tipo:

- **Casos de Uso** (`casos-de-uso/`): Diagramas que descrevem as interações entre os atores e o sistema, documentando os requisitos funcionais e os fluxos principais de uso.
- **Classes de Análise** (`classes-de-analise/`): Diagramas que detalham a estrutura estática do sistema, mostrando classes, atributos, métodos e relacionamentos.

## Como Visualizar os Diagramas

Os diagramas estão no formato `.drawio` e `.puml` e podem ser abertos com:
- [Draw.io](https://app.diagrams.net/) (online)
- Extensão do VS Code: Draw.io Integration
- Aplicativo desktop Draw.io
- [PlantUML](https://www.plantuml.com/) (online)
- Extensão do VS Code: PlantUML
